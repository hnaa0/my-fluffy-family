# My Fluffy Family
⚙ `react`, `vite`, `typescript`, `react-redux`, `styled-component`, `styled-reset`

서울시 공공데이터를 이용해 만든 반려동물 입양 홍보사이트입니다.  
웹, 모바일 버전을 제공하는 반응형 웹입니다.
